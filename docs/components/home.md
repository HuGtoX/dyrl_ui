### 介绍

@dyrl/mobile 是uniapp项目的内部组件库

### 特性

- 💪 使用 TypeScript 编写，提供完整的类型定义
- 📖 提供丰富的中文档和组件示例
- 🍭 支持 Vue 3 、 H5、APP 和微信小程序
- 🍭 支持按需引入和 Tree Shaking
